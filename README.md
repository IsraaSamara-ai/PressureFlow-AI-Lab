# 🌊 PressureFlow AI – مختبر تفاعلي لضغط الموائع والغازات مع تنبؤ حركي بالذكاء الاصطناعي  

**إعداد وتطوير: Israa Samara**  

## 📌 نظرة عامة  
هذا المشروع التعليمي الاحترافي يسمح للمستخدمين باستكشاف فيزياء ضغط المائع الساكن (`Static Fluid Pressure`) وضغط الغاز المثالي (`Ideal Gas Law`) مع تغيير المتغيرات في الوقت الفعلي. يتضمن **نموذج ذكاء اصطناعي** (Linear Regression) للتنبؤ بحركة الجسم في المائع (السرعة النهائية)، و**رسوم متحركة سلسة** باستخدام Plotly، و**أجهزة قياس ضغط افتراضية** (بوردون، مانومتر، مستشعر إلكتروني).  

## 🚀 الميزات الرئيسية  
- **حساب ضغط المائع**: `P = ρgh` مع تغيير العمق والكثافة.  
- **قانون الغاز المثالي**: `PV = nRT` مع اختيار نوع الغاز (He, N₂, CO₂).  
- **تأثير الطفو**: حساب الوزن الظاهري وقوة الطفو.  
- **AI للتنبؤ بالحركة**: توقع السرعة النهائية لجسم مغمور بناءً على عمقه، كثافة المائع، وحجمه.  
- **محاكاة متحركة**: رسم بياني متحرك لحركة الجسم نحو السطح.  
- **واجهة مستخدم جميلة**: ألوان مريحة، تخطيط عريض، تفاعل مباشر.  

## 🛠️ التقنيات المستخدمة  
- Python 3.9+  
- Streamlit (واجهة تفاعلية)  
- Plotly, Matplotlib (رسوم بيانية ومتحركة)  
- Scikit-learn (نموذج الانحدار الخطي)  
- NumPy, Pandas, SciPy
